# Compiler Construction

Building a compiler from scratch as part of the course compiler construction at the University of Bern.

## Getting started with JavaCC 7.0.13

- Download the binary (`.jar`) from: https://github.com/javacc/javacc?tab=readme-ov-file#getting-started
- Move the binary to `/usr/local/lib/javacc-7.0.13.jar`
- Run `java -cp /usr/local/lib/javacc-7.0.13.jar javacc <path_to_jj_file>`
